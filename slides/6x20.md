# MayMay's Latt Swal 6x20 Slides

## 1. Problem

Myanmar home cooks often spend too much time deciding what to cook each day.

## 2. Solution

MayMay's Latt Swal gives fast meal suggestions and ingredient-based recipe matches.

## 3. Architecture

React UI routes input through meal-planner logic, reads local recipe data, and formats output with recipe-assistant rules.

## 4. MVP Features

Random meal suggestions, ingredient search, and detailed Myanmar recipe instructions.

## 5. Claude Code Structure

The project includes `.mcp.json`, `.claude/agents/meal-planner.md`, and `.claude/skills/recipe-assistant/SKILL.md`.

## 6. Result

A deterministic, offline-capable cooking assistant that supports a complete Chapter 3 vertical slice.
